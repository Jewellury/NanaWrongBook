# B站高中数学基础打底视频调研

## 筛选口径

这次我按你的要求，只保留了**概念讲解类**资源，尽量避开了“压轴题秒杀”“模拟卷讲评”“口算最值”“技巧流刷题课”这类内容。具体筛选标准是：优先单集 **25 分钟以内**；优先**从零讲起、讲得慢、概念先行**；优先**系列化**、能稳定覆盖一整个板块；如果某知识点在主推 UP 里找不到同时满足“独立标题稳定 + 概念向 + 适合零基础”的单条，我会直接写“暂缺”，不硬凑。

对这位目前 **150 分卷 30–40 分** 的高二学生，我的总体建议很明确：  
**主启动线**用“夏老师的数学课”处理初高中衔接运算，再用“数学微课帮”的**基础 A**当概念主线；“乐乐课堂”只当**5 分钟预热/复盘**；“一数”适合放到第二阶段，等学生已经能听懂概念后再用来建立更完整的题型框架；“圆脸”最适合在学生“怎么都听不懂”的抽象概念处做**二次解释**。这个排序，基本对应了它们各自的站内数据、课程形态和站外讨论口碑。citeturn29view0turn41view0turn30view0turn8view0turn20view0turn27search17turn40search15turn6search7turn28search1turn21search0

“程度评级”我按这位学生的**直接适配度**来分：  
**S** = 可以直接当主线；**A** = 强辅助或某板块主线；**B** = 更适合二次理解，不建议一上来就当主线。

## 使用顺序建议

如果你是给这位学生做一份**真正能落地**的学习单，我建议不要平均铺开，而是按“先止血、再建概念、最后再接主线”的顺序来。

第一步，先把 **① 初高中衔接运算** 和 **② 集合/逻辑/不等式/复数** 做完。这里最优组合不是“一口气上长课”，而是：  
先用 **乐乐课堂** 这种 2–5 分钟微课做预热，再上 **夏老师** 或 **数学微课帮基础A** 的正课。这样的原因很简单：乐乐的优势是“快、直观、先让学生不害怕”，夏老师和微课帮的优势是“慢、稳、能真正把定义和步骤讲透”。citeturn16view0turn30view0turn29view0turn41view0

第二步，**③ 函数** 和 **④ 三角函数** 不要急着进一数的中档题型。对 30–40 分学生来说，先把“函数是什么”“单调、奇偶到底在说什么”“弧度制到底是什么”讲明白，比会做一两道题更重要。因此这里推荐先走 **圆脸 / 数学微课帮基础A**，等学生能复述概念后，再拿 **一数** 的“辞典 / 基础 / 易”去做第二遍框架补全。citeturn20view0turn41view0turn8view0turn25search2

第三步，**⑤ 平面向量** 和 **⑥ 概率统计** 最适合由 **数学微课帮基础A** 或 **一数辞典** 接手。原因是这两块最怕“老师讲得太花”，而微课帮和一数在这两块的标题就已经很老实：概念、基本定理、坐标表示、随机抽样、样本数字特征、古典概型，都是课本型命名。对这位学生，这种“课本型命名”其实就是最安全的信号。citeturn42view0turn4view0

## 空缺与注意事项

最需要你额外注意的是：**分式基本运算** 这一项，我这次没有硬塞一个标题进去。原因不是 B 站完全没有分式视频，而是我在这次主推的几位老师里，没有找到一条同时满足“标题稳定、独立概念课、适合零起点、又确实口碑够稳”的视频。对这种情况，硬塞一条不够合适的资源，反而会降低整张表的可用性。所以这一块我建议你把它明确标成“**回初中教材/讲义补**”，而不是假装已经找到完美视频。
