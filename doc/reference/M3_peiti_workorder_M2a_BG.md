## 一、出题提示词（照此产题，每题须过文末自检）

```
【角色与场景】
你在为一个面向"数学基础非常薄弱的高三学生"的诊断系统生产【初诊题】。
学生数学很弱（150 分卷曾只考 30-40 分），目标是从地基摸起、保底提分。
你产的不是考试题，是【地板题/探针题】——用来摸清"她到底会不会某一个最小知识点"。

【五条铁律】
1. 一题只测一个节点的判定标准（judgeCriteria）。绝不把多个知识点揉进一题——
   否则她做错了，我们无法判断是哪个点崩的。
2. 地板难度：面向能力下限。boundary 题尽量一步到位、数字干净、不绕弯。
3. 只对我给的节点产题，不许发明新节点、不许改 id。
4. 答案必须对：数学符号一律 LaTeX；每题必须给答案且必须正确。
   没十足把握就标 [存疑] 并说明，绝不硬编一个看起来对的答案。
5. 不要照抄 sampleItem，以它为锚另出新题。

【每个节点产出哪些题（按 role）】
- boundary ×1（必产）：判断"会不会"的最小题，直接对应 judgeCriteria。
- variant ×1（必产）：与本节点 boundary【同构】——同方法、同步骤，只换数字/最小情境，
  用于隔天复测。必须注明它复测的是哪道 boundary。
- concept ×1（节点偏概念理解时才产）：暴露"答案对、其实没懂"的题，附一句费曼追问。
- drill ×1~2（节点偏计算/程序熟练时才产）：同类反复练手的题。

【输出格式】逐题一条（YAML 或表）：
nodeId | role | stem(LaTeX) | answer | pairWith(variant专用) | conceptCue(concept专用) | note(存疑/备注)

【节奏】把下面 BG 地基 5 个 + M2a A 层 8 个全部做完，就停下，等我确认。
```

## 二、本批特别注意（这两组比 M1 更容易踩前置污染，务必看）

1. **定义域题（M2a-01 / M2a-03）别被"解不等式"污染**：测的是"定义域优先意识 / 会列限制条件"，
   不是测解复杂不等式。条件尽量用**一次式**（如 √(x+1)、分母 x-2），别整出需要解二次不等式才能定的定义域——
   否则她卡在不等式，你却记成"不会定义域"。
2. **分段函数题（M2a-09）必须写出具体的分段表达式**：sampleItem 那句"已知分段函数"是占位，
   你出题时要把分段函数完整写出来，再问 f(某值)。
3. **指数/对数比较（M2a-32 / M2a-42）保持同底、数值干净**：地板题用同底比较真数/指数即可，
   不要异底、不要中间量法（那是 C 层的事）。
4. **BG 地基题就是最朴素的初中/高一基本功**：BG100 韦达直接套公式、BG101 解一元二次不等式数字要干净、
   BG102 三种写法互转、BG103 一眼能看出换元结构、BG104 闭区间求最值轴在区间内即可（先不搞含参讨论）。

## 四、出题 AI 自检清单（每题逐条自答，不过就重出，别交上来）

```
1. 这道题只测【一个】节点的判定标准吗？测的是哪个点？（说不出唯一一个点 → 重出）
2. 她会不会因为某个【前置技能】不会而做错这道题？（会 → 把前置回避或换简单数）
   —— 定义域题尤其检查：是不是要先解一个难不等式？
3. 我亲自把答案算了一遍，确定对吗？区间开闭/正负号/根号/对数底数都核了吗？（没把握 → 标 [存疑]）
4. 难度是不是"地板级"，适合曾考 30 分的学生？（偏难 → 降到能力下限）
5.（variant）我能用 boundary 的解法一字不改走通这道 variant 吗？数值换了、方法没换？
6. 题面不靠看图就能做吗？符号是合法 LaTeX 吗？
7. 这题是不是照抄了 sampleItem？（是 → 重出一道新的）
```
