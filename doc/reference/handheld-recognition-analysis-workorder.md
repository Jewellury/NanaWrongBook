# 工单 · 手持拍照识别问题分析 + 采集/预处理建议

> 基于 `doc/research/vision-samples/handheld-report.md`（20 张真实照片，豆包 Pro 识别结果）。
> **关键前提（必须守）**：项目 AI **看不到图**，只能分析**输出文本信号**，**判不了"识别准不准"**——准确率的最终判据是孩子的 Part B 打分 + 参谋长看图。所以本工单只让你做"**输出信号挖掘 + 假设级建议**"，**严禁下"识别准确率 X%"这类越界结论**。

## 任务 1：失败/异常信号分类（基于 20 条输出）
逐图扫输出，产出一张表：`图号 | 信号 | 疑似原因（标"待看图确认"）`。信号至少含：
- "无法识别" / 空答案 / 空分析（如图 02、12）
- 局部"内容不完整 / 缺失 / 无法识别"（如图 08 第(2)问、图 15"定义域内容缺失"、图 16 从"(2)"开始）
- 多题挤在一张（如图 08、11、13、19）
- 答案/分析 = "无" / "图片未提供标准答案"
- token 异常（过低 = 内容少；过高）

## 任务 2：把信号归成三类问题（这步最关键，决定解药方向）
**必须把问题分成三类，别混**：
- **A. 识别问题**（模糊/手写读错/公式错）→ **你判不了，全部标"待看图/待打分"**。
- **B. 采集问题**（裁切不全、多题混张、拍歪）→ 可从"不完整/缺失/从半句开始/一张多题"信号推断。
- **C. 内容本就缺失**（教辅题页本就没标准答案、她没写解答）→ 可从"图片未提供标准答案"类信号推断。
> 区分意义：A 的解药是图像预处理 / 换提示词；B 的解药是**改采集引导**；C 的解药是**改流程**（答案/解答从哪来）。

## 任务 3：建议（假设级，标依据与不确定）
- **针对 B（采集）**：给具体采集引导建议，用于更新拍照说明（`photo-guide-workorder.md` 产出的文档）——例如拍全别裁、拍正、一题一拍 or 多题须拍全、答案/解答页另拍。
- **针对 A（识别）**：**先别急着上预处理**。建议"失败驱动"——等参谋长看图/孩子打分确认确有识别问题，再针对性补对应预处理（透视/去阴影等），不预先堆。
- **针对 C（内容）**：建议流程上怎么处理"答案不在照片里"（让她写/另拍答案册/标注无答案）。

## 任务 4：一个判断题（请明确回答 + 依据）
从这 20 条输出看：**豆包 Pro 的"识别能力"像不像瓶颈？还是瓶颈更在"采集方式 + 答案可得性"？** 给出判断和依据（基于信号，不越界）。

## 分工（写清，别越界）
- **项目 AI（你）**：输出信号分析（看不到图，只做 A/B/C 分类 + 假设级建议）。
- **参谋长（外部）**：看图 spot-check 异常图（02 / 08 / 12 / 15 / 16 等），确认真实根因，回填到你这份报告的"待看图确认"处。

## 参谋长看图核定（回填，2026-06）—— 以此为准更新提示词
参谋长已逐张看图核定 5 张异常图。结论：**豆包 Pro 识别不是瓶颈，"异常"几乎全是采集/内容问题，纯识别能力失败 0 张。**

| 图 | 真实根因 | 类别 |
|----|---------|------|
| 02 | 目标题=第12题（满屏红蓝手写解答），顶部夹带第11题答案尾巴 | 邻题夹带（非识别失败）|
| 08 | 目标题=第11题（有手写解答），底部夹带残缺的第12题 | 邻题夹带 |
| 15 | 目标题=第13题（有手写解答），顶部夹带残缺的第12题 | 邻题夹带 |
| 16 | 只拍到大题的一部分，题号 +(1) 在画面外 | 采集没拍全 |
| 12 | 她只写了第(1)问第一步就没继续做 | 内容未作答（非识别/采集失败）|

**核心发现：目标题 = 有手写作答/涂改的那道；边缘只有印刷、无手写的残题 = 夹带进来的邻题，应忽略。**

### 据此更新 handheld 识别提示词（必做）
在 `scripts/vlm-handheld-test.ts` 所用提示词里加一条**目标题锁定规则**：
> 一张照片里若出现多道题，**以"含手写作答/批改痕迹"的那道为目标题**，完整提取它即可；**边缘只有印刷题干、没有手写的残缺邻题，请忽略——不要单独提取、不要报"内容不完整"。**

加上这条后，02/08/15 那类"内容不完整/缺失"噪声会直接消失。

### 据此修正 A/B/C 定性
原先标的"B 类裁切（把目标题切掉）"**更正为"邻题夹带（噪声，非目标题失败）"**——目标题本身都识别成功了。最终：邻题夹带 3、采集没拍全 1、未作答 1、**纯识别能力失败 0**。这进一步证实「不需要图像预处理算法，重点在采集引导 + 提示词锁定目标题」。
