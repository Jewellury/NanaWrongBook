# M3c 工单 · 周末 session 编排 + 最小纸质包 PDF

> 性质：给项目 AI 的**范围工单**。拿它去走 `/plan`，参谋长审过 plan 再 `/execute`。
> 一句话目标：把已有零件（M2 状态机 + M3a 初诊/BKT/地图）**串成一次能跑的诊断流程**，
> 并给一个**看得见结果的最小出口**——周中可打印的纸质包 PDF。
> 主旨：现在最大风险仍是"她到底会不会用"。这轮不是堆引擎，是让系统**第一次端到端可用**。

---

## 一、本轮要做（IN）
1. **初诊编排 + 结果持久化**
   - 给定 `studentId + mainlineId`：取该主线 A 层节点的 **boundary 题**（Item 表，role=boundary，决策⑧）→ 组成"题单"返回。
   - 接收答案（`correct` 布尔，**人工判分**，决策⑦）→ 跑 KST-lite + BKT → **把结果写入 `StudentNodeState`（M1 既有自建表）**。
   - 用 M2 的 `DiagnosisSession` 记录这次会话；用 M2 `SessionMachine` 串相关步骤。
2. **最小纸质包 PDF**
   - 为"学习前沿 + 近期 gap 节点"从 Item 库选**练习题**（variant/drill），生成一张**可打印**的周中练习单。
   - PDF 含：封面（她的名字 + 本周 + 一句鼓励）、练习题（留手写空间）、**答案页分开**（给大人对）。

## 二、本轮不做（OUT，防膨胀）
- ❌ 不调 LLM / AI（不自动判分、不生成 Newman 追问、不 AI 解析）
- ❌ 不做花哨 Web UI（交互式地图可视化、周末上屏体验 = 另一个 UI 轮）。本轮"出口"就是 PDF。
- ❌ 不做探针下探自动化（M4）
- ❌ 不做跨主线 / BG 下探（沿用单主线初诊，决策⑥）
- ❌ 不把 M2 状态机 8 步全软件化——只走通"初诊→更新地图→出纸质包"这几步的数据落地

## 三、关键决策（请 /plan 把这些钉进去，别漂移）
- **A. `StudentNodeState` 必须持久化（本轮命门）**：初诊结果要写库、跨会话可读——这才是"持久知识状态地图"。
  /plan 先**确认 M3a 当前是否已持久化**；若只是 return 没落库，本轮补上。
- **B. 判分仍人工**（决策⑦延续）：题单/PDF 给题，对错由舅舅/她录入，系统不自动判分。
- **C. 纸质包选题**：boundary 已用于初诊；纸质包用 **variant（隔天复测）+ drill（练熟练）**——这俩终于派上用场。
  **concept 不进自动流**（决策⑧），纸质包也不放 concept。
- **D. P4 后台化（措辞铁律）**：纸质包/题单**只呈现"要练什么"和绿点进展，绝不报"你又错了"、不铺一片红**。
  文案正向。可把周中三条极简规则（过程写题旁不擦、做不出贴标签、卡住画箭头）印在单子上。
- **E. PDF 选型先查既有能力**：wrong-notebook **本身自带"错题导出打印 PDF"功能**——/plan **必须先去翻既有 PDF 导出实现，优先复用/参考，不要另起一套**。确有不足再增量补。
- **F. 是否新增表由 /plan 定**：如需记"本周发了哪份纸质包"可新增一张挂接表（不碰上游）；最小实现也可不建表、按需生成。给出理由。

## 四、测试策略（新规已生效）
- **选题逻辑、PDF 内容组装** = 逻辑重 → **测试先行**。
- PDF 渲染、API 路由 = 样板 → 后置。
- 集成测试：真实 handler + mock session；测试容器跑、退出码 0、dev.db 隔离；新增 `test:*` 同步进 `test:all`。

## 五、安全铁律（照旧）
- 新增表/字段一律新增挂接，**不碰 wrong-notebook 上游**；`StudentNodeState`/M2 表是我们自建的，写它 OK。
- 需要 migrate → **铁律 1 先确认**（确认 diff 只新增、不 ALTER 既有上游表）。
- DB 护栏、测试容器、test.db 隔离照旧；测试"写了≠过了"。

## 六、验收方向（/plan 细化成清单）
- 能跑通一次：建 session → 拿到某主线 boundary 题单 → 录入答案 → **`StudentNodeState` 落库** → 地图 API 读得到更新后的状态。
- 能生成一份纸质包 PDF：选题来自 variant/drill、含答案分页、措辞正向、可打印。
- 全测试在安全路径绿、退出码 0。
